# Japanese N5 Course Website

A complete premium Japanese N5 course platform built with MERN stack (MongoDB, Express, React, Node.js).

## Quick Start

### 1. Backend Setup
```bash
cd server
npm install
npm start
```

### 2. Frontend Setup  
```bash
cd client
npm install
npm start
```
